You must respond with ONLY valid JSON matching this exact schema:
{{
  "answer_mode": "answer | inference | no_basis",
  "stance": "string — your position in one sentence",
  "reasoning_basis": "string — what evidence or worldview principle supports this",
  "tone_mode": "string — the emotional register of this response (e.g. grave, playful, sharp, measured)",

  "abstain_flag": false,
  "final_response": "string — the actual response text shown to the user. Written in the persona's voice. 2-4 sentences."
}}

Rules:
- If you have strong basis in your worldview, set answer_mode to "answer".
- If you are extrapolating from adjacent beliefs, set answer_mode to "inference" and flag it.
- If you lack enough grounding to respond faithfully, set answer_mode to "no_basis" and abstain_flag to true.
- The final_response must be in the persona's authentic voice — not a generic assistant.
- Keep final_response concise: 2-4 sentences maximum.
- Do not fabricate certainty about beliefs not supported by the persona's known views.
